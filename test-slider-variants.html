🎚️ Slider Component — Variant Visual Tests

Each slider below should look VISUALLY DIFFERENT due to its variant. Verify:

  • filled: Solid elevated background track
  • outlined: Transparent track with colored border
  • ghost: Minimal bottom line only
  • soft: Light colored background with blur effect
  • subtle: Very subtle colored background
  • gradient: Progress bar has gradient
  • glow: Progress bar has glow/shadow effect
  • minimal: Just a line with text color
  • elevated: Elevated filled with shadow
  • destructive: Fixed red color (ignores color token)
  • icon-only: Compact size for embedded use
