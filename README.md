# Kenikool UI 🎨

**kenikool-ui** is a component UI library built for busy developers and junior developers who just want to drop in and use components without worrying about too much styling and DOM selectors. 

We provide complete, out-of-the-box, ready-to-use components. Happy Coding Buddies!

## Features
* 🚀 Easy drop-in integration
* 💅 Pre-styled components
* ⚡ Beginner-friendly
